# Photos du site BCV — où déposer les fichiers

Pour ajouter une photo : dépose le fichier **au bon endroit avec le nom exact**
ci-dessous, puis `git add`, `git commit`, `git push`. La photo apparaît au
prochain déploiement — **aucune modification de code nécessaire**. Tant qu'un
fichier n'existe pas, un placeholder « Photo à venir » s'affiche automatiquement.

> Conseil : redimensionne chaque photo à **~1600 px de large maximum** (format
> `.jpg`) avant de la déposer, pour ne pas alourdir le site.

## Coachs — `images/club/coachs/coach-N.jpg`

`coach-1.jpg`, `coach-2.jpg`, `coach-3.jpg`, `coach-4.jpg`
(ordre = ordre d'affichage sur la page Le Club).

## Bureau — `images/club/bureau/[role].jpg`

`president.jpg`, `secretaire.jpg`, `tresorier.jpg`

## Actualités — `images/actus/[slug]/…`

Un sous-dossier par article (nommé selon le slug de l'article), créé au besoin.

## Uploads Decap CMS — `images/uploads/`

Dossier où le CMS (`/redaction`) dépose les images ajoutées via l'interface web.
Ne pas y toucher manuellement.
